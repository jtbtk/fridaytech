Flexbox est un modèle de disposition unidimensionnel qui permet de gérer l'alignement, l'espacement, l'ordre et la taille des éléments selon un axe principal qui peut être horizontal ou vertical.

Flexbox propose des propriétés CSS spécifiques pour le conteneur flex parent et les éléments enfants flexibles.

Propriétés pour le conteneur flex

Pour définir un conteneur flex, il faut utiliser la valeur flex à la propriété display.

Définir l'axe principal et transversal

La propriété flex-direction permet de définir l'axe principal et l'axe transversal d'alignement dans une flexbox.
