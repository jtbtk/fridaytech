Le compilateur JIT (Just-In-Time) de Tailwind CSS permet d'améliorer les performances en réduisant la taille du fichier CSS généré. Il accélère également le temps de chargement du site et facilite le développement en compilant uniquement les classes utilisées dans le projet, ainsi que celles incluses dans la safelist.

Le compilateur permet d'utiliser des valeurs arbitraires dans les noms des classes.
