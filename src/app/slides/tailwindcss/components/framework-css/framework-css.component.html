Un framework CSS est une collection de styles CSS pré-définis qui permet de faciliter la création d'interfaces utilisateur modernes.

Les frameworks CSS sont généralement utilisés avec du code HTML et JavaScript pour créer des interfaces utilisateur cohérentes et réactives.

Component-based CSS frameworks

Ils fournissent des composants prêts à l'emploi tels que des boutons, des cartes et des menus, ... Ils peuvent également fournir des classes utilitaires pour personnaliser leur style.

Utility-First CSS frameworks

Ils fournissent principalement un ensemble complet de classes utilitaires.

Une classe utilitaire est une classe qui définit un style spécifique et généralement basique. Elle peut être appliquée seule ou en combinaison avec d'autres classes pour personnaliser le style des éléments HTML.
