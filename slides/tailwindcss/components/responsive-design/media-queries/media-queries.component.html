Les media queries permettent de personnaliser le style des éléments HTML en fonction des caractéristiques de l'appareil utilisé, comme la taille de l'écran, l'orientation de l'écran et le type de média.

Les opérateurs logiques or, and et not peuvent être utilisés pour combiner des conditions dans les règles @media.

L'opérateur logique only peut être utilisé pour spécifier que la règle ne s'applique que si toutes les conditions spécifiées sont remplies et que le périphérique de sortie correspond à la liste de médias spécifiée.

Media queries en pratique

Cibler les appareils en fonction de la taille de leur écran

Utilisation de width, min-width, max-width pour définir des styles pour différents points de rupture.

Point de rupture : seuil de déclenchement.

Cibler les appareils en fonction de l'orientation de leur écran

Utilisation de portrait ou landscape pour définir des styles pour une orientation d'écran.

Cibler les appareils en fonction de leur type de média

Utilisation pour définir des styles pour certains types de média tels que all pour tous les types d'appareils, print pour les imprimantes, screen pour les appareils avec un écran, speech pour les lecteurs d'écran, ...
