Le fichier de configuration peut être utilisé pour :
  • étendre des sections du fichier de configuration par défaut avec extend
  • remplacer des sections du fichier de configuration par défaut
  • ajouter des plugins à la section plugins pour étendre les fonctionnalités de Tailwind CSS
  • personnaliser certaines autres options de Tailwind CSS : !important, un préfixe aux classes générées, ...
