Tailwind CSS est un framework CSS utilitaire personnalisable qui fournit des classes utilitaires organisées par catégories et nommées de manière significative.

Le but de Tailwind CSS est de faciliter la conception d'interfaces utilisateur modernes sans avoir à écrire de CSS personnalisé et sans quitter le code HTML.

Tailwind CSS s'intègre parfaitement avec de nombreux autres outils de développement tels que Angular, React, Laravel, ...
