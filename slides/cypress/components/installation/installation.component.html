cypress run

La commande permet d'exécuter tous les tests de manière séquentielle en utilisant un navigateur headless (par défaut) basé sur Chromium, qui simule un environnement de navigateur réel. Cette méthode garantit que les tests s'exécutent de manière cohérente, fiable et plus rapide, quel que soit l'environnement de l'utilisateur final. Cette fonctionnalité est utile pour exécuter des tests de manière automatisée dans un pipeline CI/CD.

Quelques options qui peuvent être ajoutées à la commande npx cypress run:

npx cypress run --headed

npx cypress run --browser chrome

npx cypress run --spec cypress/integration/my_test.spec.ts
